# Attend - Student Attendance Platform

An efficient and easy-to-use application designed to help educators track attendance in real-time, analyze excuses with AI, and identify at-risk students.

## What problem does it solve and for whom?
It solves the time-consuming problem of manual attendance tracking for educators. It provides real-time insights and automated analytics to help manage classrooms effectively without spreadsheets or paper rosters.

## Features List
- **Attendance Dashboard**: Mark attendance in seconds, bulk mark entire classes, and view 30-day trend charts.
- **AI Excuse Analyzer**: Automatically score parent notes or emails against school policy to determine validity with confidence scores.
- **Trend Tracking**: Monitor daily attendance rates and view comprehensive class trends.
- **Class Management**: Easily create classes, manage rosters, and record daily summaries.

## AI Feature
The **AI Excuse Analyzer** evaluates parent notes against school policies to provide verdicts (Valid/Invalid/Needs Review), reasoning, and suggested actions instantly.
